# PowerShell

Adding PowerShell to Visual Studio Code's `settings.json`

```json
    ,
    "terminal.integrated.profiles.windows": {
        "PowerShellCore": {
          "path": ["C:\\Apps\\PowerShell\\7\\pwsh.exe"],
          "args": []
        }
    },
    "terminal.integrated.defaultProfile.windows": "PowerShellCore"
```
